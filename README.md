# Linux

## Basic Commands

- ls
    - List files : ls
    - List files with details : ls -l or ll
    - List hidden files : ls -a
- touch : to create empty files/change timestamp of files
    - create file using touch : touch f1
- cat :
    - Create new file : cat > f1
    - cat > f1
- open a file using cat : cat f1
